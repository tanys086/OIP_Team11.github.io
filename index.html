Measuring Kibble Palace Footfall with LiDAR Technology

Helping the Kibble Palace accurately count visitor numbers using LiDAR sensing technology that respects the building's heritage status while providing detailed analytics.

Project Overview

Stakeholders

This project is in collaboration with the Friends of Glasgow, with the goal of coming up with a solution that provides accurate footfall analytics for the Kibble Palace.

The Challenge

The Kibble Palace, a 19th-century glasshouse and Category A Listed building, lacks accurate visitor counting systems.

Our LiDAR Solution

A non-invasive LiDAR-based visitor counting system that provides real-time analytics without physical modifications to the building.

LiDAR Sensor

Strategic placement of the LiDAR sensor helps accurately detect and count visitors without requiring structural modifications.

Budget Friendly

Using Raspberry Pi and low-cost LiDAR sensors, we developed a budget-friendly solution that meets financial constraints.

Heritage Compliance

Designed with the building's heritage status in mind, ensuring no physical alterations while tracking visitors effectively.

Data Visualization

LiDAR data is sent to the interactive dashboard via Raspberry Pi for real-time monitoring and visitor analytics.
